# vue-admin

vue管理系统模板

## 实现



### 后端



* koa全家桶
* token验证
* mongodb数据库



###  前端





* 路由重定向
* 路由钩子检验权限
* vuex链接localstorage存取token等数据
* 组件懒加载
* vuex管理路由表
* router.addRoutes动态添加路由





### 打包



* 打包分析
* `DLLPlugin` 和 `DLLReferencePlugin` 拆分 bundles，同时还大大提升了构建的速度

* vue-cli3的webpack配置
* inspect 分析 webpack 配置文件
* AddAssetHtmlPlugin引入dll
